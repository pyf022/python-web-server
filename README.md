# Python AI Tool Server

一个面向 AI agent 和第三方系统的工具 API 网关。第一版包含文件上传下载、API Key 认证、同步/异步工具调用、Redis/RQ worker、可选 Webhook，以及常用文档转换和文件处理工具。

## Quick Start

```powershell
docker compose up --build
```

API 文档：

- Swagger UI: http://localhost:8000/docs
- OpenAPI JSON: http://localhost:8000/openapi.json

默认 API Key:

```text
dev-api-key
```

调用时添加：

```http
X-API-Key: dev-api-key
```

## Example

上传文件：

```bash
curl -H "X-API-Key: dev-api-key" -F "file=@README.md" http://localhost:8000/v1/files
```

同步 Markdown 转 PDF：

```bash
curl -H "X-API-Key: dev-api-key" \
  -H "Content-Type: application/json" \
  -d "{\"input\":{\"markdown_text\":\"# Hello\\n\\nGenerated by agent.\"}}" \
  http://localhost:8000/v1/tools/md_to_pdf/run
```

异步任务：

```bash
curl -H "X-API-Key: dev-api-key" \
  -H "Content-Type: application/json" \
  -d "{\"input\":{\"text\":\"{\\\"b\\\":1,\\\"a\\\":2}\",\"format\":\"json\"}}" \
  http://localhost:8000/v1/tools/json_yaml_format/jobs
```

## Notes

- Docker 镜像内安装 LibreOffice、Pandoc、wkhtmltopdf，用于高保真转换。
- 本地无系统转换工具时，`md_to_word` 和 `md_to_pdf` 会使用纯 Python fallback。
- `pdf_to_word` 是尽力转换，扫描版 PDF 需要后续接入 OCR 扩展。
